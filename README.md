An Artificial Intelligence project that uses Python, LLMs, RAG, LLaMA 3.2, Gemma 3, LangChain, Ollama and Chroma, capable of functioning as a local chatbot that answers pizza related questions using a vector database built from CSV based reviews.

1. Install Ollama on your computer, running on port 11434
2. Download at least the following models: llama3.2:latest, gemma3:latest
