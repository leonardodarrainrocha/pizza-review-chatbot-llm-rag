# RAG Project – Earlier Iteration

This project represents an earlier iteration of a Retrieval-Augmented Generation (RAG) system built using higher-level tooling.  
My current work focuses on building similar systems from scratch with greater control, modularity, and production readiness.

## Project Overview

This Artificial Intelligence project demonstrates a local chatbot capable of answering pizza-related questions.  
It leverages:

- Python
- LLMs
- RAG frameworks
- LLaMA 3.2, Gemma 3
- LangChain, Ollama, and Chroma

The system uses a vector database built from CSV-based reviews to generate context-aware responses.

## Setup Instructions

1. Install [Ollama](https://ollama.com/) on your computer, ensuring it runs on port `11434`.  
2. Download at least the following models:
   - `llama3.2:latest`
   - `gemma3:latest`
